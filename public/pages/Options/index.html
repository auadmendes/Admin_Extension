Admin Console Extension

A guide to use the extension

The purpose of this project

The purpose of this project is to help in the agent's daily work. With custom views and fields, in addition to reducing clicks and searches, providing efficient ways to copy and access information.

Downloading the extension

This extension will work locally only, to respect the Trustly security. Having access to the extension folder you must download the dist folder.

dist

Save it in a local you will not delete it by accident.

How to install

Open your browser go to the toolbar as you can see below click on the icon for extensions and then Manage extensions.

Or you can click on the 3 dots, click on extensions, and then Manage extensions

Within the page opened you must turn on the developer mode.

Clicking on that will enable you to load your extension. Click on the Load unpacked and select your extension.

Select the Dist folder you downloaded from the link provided and click Select.

Enable your extension by clicking on the Switch button as you can see below.

Go to your Admin Console and refresh the page.

How to use the extension

Click on the Extension button and pin the extension.

After refreshing your Admin Console with the Extension enabled your field view is the same.

To choose what fields you want to display click on the Extension’s icon, and click on the right arrow button.

The Fields have the same name as you can see in Admin Console, click to select or unselect them, they will appear in the order you choose them.

The anatomy of the transactions page

  • Improved aspect.
  • The logo was changed to Trustly’s one.
  • Fields can be enabled/disabled on the page’s header.
  • The buttons for Filter and Columns have a new style.
  • A box with a checkbox count and a span to display the sum of the amount was added.
  • The button reset received a new style.
  • On the table, the second column Trx ID was enabled to receive a single click and open a new page.
  • The Merchant Reference received a link option, we can click on that to open the transaction.
  • The Action button received 6 new options.

Fields

When it is your first time using the Admin Extension your fields are the same as you have been using all this time, if you are comfortable with that you don't need to change it. If you want to, you must map that by clicking on Map Fields at the bottom of the extension pop-up.

It will delete your field view and leave it blank. You must now select what you would like to be displayed.

Total count

By selecting any transaction with the checkbox you will see how many transactions checked you have and the sum of the amounts.

Action button

By selecting transactions you can click on the Action button and click on the Selected PTX option to copy its PTX, it helps to create Proof of authorization.

You can do the same with the Merchant Reference option or copy all the PTXs or all the Merchant references by clicking on All PTXs and All Merchant Ref.

Generating POA

For the Back office team, we do POA (Proof of authorization) every day, multiple times. To generate the POA from the transactions page select all the transactions you want to add to the POA, and click on the Action button, after that click on the Generate POA option.

This option will create a new tab, copy all the information, and place them into the POA page. You must click on Add Payments at the bottom of the page.
