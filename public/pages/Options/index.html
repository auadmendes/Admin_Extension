Admin Console Extension

A guide to use the extension

Fields displayed

WHen you open Admin Console, you can see all the fields that are common to any user. You can remap it by clicking on the extension icon.

Go to the extension icon on the top of your browser, if you have not, you can pin it, Click on it and a pop up will be presented, on the top of that you can click on the button > No you have a button to Map Fields, it will read and save all the fields, including the ones under More Filters. Reload your Admin page, all the displayed fields are cleared. Click on the extension go to >. You can see now all the fields, click on each fields you want to display, it will be added in the order you click on that.
